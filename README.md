# Catloops Cat Cafe

A membership management web app built for Catloops Cat Café in Kamloops, BC. 
Members can sign up and select a membership tier online, and staff manage 
approvals through an admin dashboard.

## Features

- Member registration with student, senior, and regular tier selection
- Automated pricing based on membership category ($60 or $75/month)
- JWT authentication with role based access control
- Admin dashboard to view and activate pending memberships
- E2E tested with Playwright covering core user flows
